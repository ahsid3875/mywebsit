Fibonacci Series Flowchart

flowchart TD A([Start]) --> B[Initialize a = 0 and b = 1] B --> C[Input n (number of terms)] C --> D[Set i = 0] D --> E{Is i < n?} E -->|Yes| F[Print a] F --> G[Set next = a + b] G --> H[Set a = b] H --> I[Set b = next] I --> J[Increment i by 1] J --> E E -->|No| K([End])
